# Slackers
Knowledge on slack space or know how to use a hex editor

## Question Text
Tom has been slacking in class, he needs help to find the answer

## Distribution
Distribute all the contents inside `distrib` folder to the users.

## Solution
1. Mount the iso image on your linux machine and inspect the slack space using a program like bmap.  
(Guide on how to use bmap here: http://computersecuritystudent.com/FORENSICS/HIDING/lesson1/)  
2. or open the iso in a hex editor and find 'GCTF{' from the end of file up.
